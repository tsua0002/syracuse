# Syracuse Exploration Plan

## Goal

Implement the Syracuse/Collatz function, verify it with tests, generate visual outputs for individual and grouped sequences, produce a summary table, then use those outputs to test simple numerical hypotheses.

## Steps

1. Set up Pipenv dependencies for plotting and testing.
2. Implement reusable sequence, statistics, table export, and plotting functions.
3. Add a command-line script that generates outputs for a configurable range.
4. Add tests covering the function, the sequence for 11, and aggregate statistics.
5. Generate outputs for the first 1000 positive integers.
6. Formulate and test initial hypotheses using the generated dataset.
